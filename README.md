## 🚀 Getting Started

Clone and run locally:

```bash
# Clone the repository
git clone https://github.com/HimanshuSingh1531/Himanshu-portfolio.git
```

```bash
# Navigate into the project folder
cd portfolio
```

```bash
# Install dependencies
npm install
```

```bash
# Start the development server
npm run dev
```
